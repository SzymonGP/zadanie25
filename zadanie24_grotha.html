Temat:Właściwości czcionek-teoria.

  1. Jakie standardowo dostępne są czcionki w systemie Windows?

    • Times New Roman (szeryfowa)
    • Arial (bezszeryfowa)
    • Courier New (monotypiczna)

  2. Jakich czcionek dodatkowo dostarcza Internet Explorerem.

    • Verdana
    • Tahoma
    • Trebuchet MS
    • Georgia

  3. Wymień rodziny ogólne.

    • serif - czcionka szeryfowa
    • sans-serif - czcionka bezszeryfowa
    • monospace - czcionka o stałej szerokości znaków - monotypiczna
    • cursive - czcionka mająca pewne cechy pochyłej
    • fantasy - czcionka fantazyjna
      •

  4. Kiedy stosujemy rodziny czcionek, w jaki sposób wpisujemy.

    • Rodziny ogólne wykorzystuje się jako ostatnia alternatywa - jeśli użytkownik nie
      będzie posiadał żadnej czcionki, której nazwę wpisaliśmy wprost.
    • body
      {
      font-family: Arial, Helvetica, Verdana, sans-serif;
      }

  5. Rodzaje wyrównanie tekstu .

    • "left" - wyrównanie tekstu do lewego marginesu (domyślnie)
    • "right" - wyrównanie tekstu do prawego marginesu
    • "center" - wyśrodkowanie
    • "justify" - wyrównanie do obu marginesów jednocześnie (justowanie)

  6. Kolor tekstu.

    • Dzięki temu poleceniu można ustalić dowolny kolor tekstu.
    • Składnia reguły:
      selektor {color: kolor;}

  7. dekoracja tekstu.

    1. Podkreślić - (underline)
      selektor

      {
      text-decoration: underline;
      }

    2. Przekreślić - (line-through)
      selektor

      {
      text-decoration: line-through;
      }

    3. Nadkreślić - (overline)
      selektor

      {
      text-decoration: overline;
      }

    4. Można dodać migotanie tekstu (tylko Netspace/Mozilla/Opera) – (blink)
      selektor

      {
      text-decoration: blink;
      }

  8. Warianty czcionek.

    • normal - czcionka normalna (podstawowa)
    • small-caps - kapitaliki

  9. Pogrubienie czcionek.

    • normal - czcionka normalna
    • bold - czcionka pogrubiona
    • 100, 200, 300, 400 (odpowiednik "normal"), 500, 600, 700 (odpowiednik "bold"), 800,
      900 - każdy następny numer wskazuje wagę czcionki przynajmniej tak samo
      wytłuszczonej jak dla poprzedniej wartości w sekwencji

  10. Rozmiar czcionek.

    • xx-small - najmniejsza
    • x-small - mniejsza
    • small - mała
    • medium - średnia
    • large - duża
    • x-large - większa
    • xx-large - największa

  11. Transformacja tekstu.

    • Składnia reguły:

      selektor
      {
      text-transform: transformacja;
      }
    • "none" - bez zmian
    • "capitalize" - zamiana pierwszych liter wszystkich wyrazów na wielkie
    • "uppercase" - zamiana wszystkich liter na wielkie
    • "lowercase" - zamiana wszystkich liter na małe

  12. Wcięcie tekstu .

    • Składnia reguły:

      selektor
      {
      text-indent: wcięcie;
      }

    • piksele - px
    • punkty edytorskie - pt
    • wysokość litery "x" - ex
    • wysokość aktualnej czcionki - em
    • centymetry lub milimetry - cm, mm
    • części dziesiętne centymetra pisane z kropką np.: 0.5cm
    • cale - in (1in=2.54cm)
    • procent wielkości nadrzędnej - %

  13. Odstęp miedzy wierszami.

    • Składnia reguły:

      selektor
      {
      line-height: odstęp;
      }

    • piksele - px
    • punkty edytorskie - pt
    • wysokość litery "x" - ex
    • wysokość aktualnej czcionki - em
    • centymetry lub milimetry - cm, mm
    • części dziesiętne centymetra pisane z kropką np.: 0.5cm
    • cale - in (1in=2.54cm)
    • procent wielkości nadrzędnej - %
    • "normal" - przywraca wartość domyślną

  14. Odstęp między wyrazami.

    • Składnia reguły:

      selektor
      {
      word-spacing: odstęp;
      }

    • piksele - px
    • punkty edytorskie - pt
    • wysokość litery "x" - ex
    • wysokość aktualnej czcionki - em
    • centymetry lub milimetry - cm, mm
    • części dziesiętne centymetra pisane z kropką np.: 0.5cm
    • cale - in (1in=2.54cm)
    • "normal" - przywraca wartość domyślną

  15. Odstęp między literami.

    • Składnia reguły:

      {
      letter-spacing: odstęp;
      }

    • piksele - px
    • punkty edytorskie - pt
    • wysokość litery "x" - ex
    • wysokość aktualnej czcionki - em
    • centymetry lub milimetry - cm, mm
    • części dziesiętne centymetra pisane z kropką np.: 0.5cm

  16. Łączenie właściwości czcionek i tekstu.

    • Składnia reguły:

      selektor
      {
      font: wartości atrybutów;
      }

    • Jako wartości atrybutów wpisać należy zgodnie z podaną poniżej kolejnością, oddzielając
      spacjami, konkretne wartości, które opisują poszczególne atrybuty czcionek i tekstu.
      Można niektóre pominąć.
    • wartości atrybutów:

    • font-style
    • font-variant
    • font-weight
    • font-size
    • line-height
    • font-family

    • Dzięki temu poleceniu można w jednym miejscu zebrać atrybuty dotyczące czcionek i
      tekstu. Pomiędzy font-size, a line-height należy wstawić ukośnik (bezpośrednio przed
      line-height).
